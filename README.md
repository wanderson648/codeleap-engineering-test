How to Install
=================

- To download the project follow the instructions bellow:

```bash
1. git clone
2. cd codeleap-engineering-test
```

- Install the dependencies and start:

```bash
3. yarn
4. yarn dev
```
- or

```bash
3. npm install
4. npm run dev
```
